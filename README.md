# Blockbuster-DH

application developed with laravel (version 5.8)

## How to install

```sh
$ composer install
```

## How to run the migrations

```sh
$ php artisan migrate
```

## How to run the seeders

```sh
$ php artisan db:seed
```

## How to create a key

```sh
$ php artisan key:generate
```

## How to start the application

```sh
$ php artisan serve
```

### Listening port on 8000
```sh
$ http://localhost:8000/
```
